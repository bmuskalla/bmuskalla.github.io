Committer

  • ◇ Gradle Build Tool — Gradle is a build tool with a focus on build automation and support for multi-language development.
  • ◇ JDT Call Hierarchy - Filter Test Code — Eclipse plugin to show and hide callers from tests in the call hierarchy
  • ◇ vscode-tldr — Provides tldr hover information for common commands in bash/dockerfiles/powershell files.
  • ◇ Atom+Go - Jump to Test — Easily jump between Golang source code and the corresponding test code.
  • ◇ Eclipse Mylyn — Mylyn makes tasks a first class part of the IDE, integrates rich and offline editing for ALM tools, and monitors your programming activity to create a “task context” that focuses your workspace and automatically links all relevant artifacts to the task-at-hand.
  • ◇ Eclipse EGit — EGit is an Eclipse Team provider for the Git version control system.
  • ◇ Eclipse Marketplace Client — Marketplace Client is a rich client solution for installing solutions listed on Eclipse Marketplace directly from an Eclipse.
  • ◇ Eclipse Remote Application Platform — RAP allows single-sourcing existing Rich Client Applications (RCP) to be used on the web.
  • ◇ PEAR Package - DB_eXist (inactive) — Created DB_eXist PHP library to access the native XML database eXist.
  • ◇ Mylyn Connector for Google Code (inactive) — This Eclipse plugin is a Mylyn connector for Google Code. It uses the GData Java Client Library.
